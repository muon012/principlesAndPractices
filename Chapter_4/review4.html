Terms

  1. Abstraction
      An important tool for organizing a program it is the act of hiding details or unnecessary(unnecessary to the target audience like an user of a program) data behind easy to use models/GUI/tools etc.
  2. Computation
      It is the act of producing output(result) from some sort of input. This could be from an user to a computer, or even from a program to another program.
  3. Conditional Statement
      A statement that contains an expression which result could be true or false.
  4. Declaration
      It is stating the name of a variable and its type. No value has been given to it, so it is not 'defined'.
  5. Definition
      Assigning a value to a previously declared variable. Initialization is declaration and definition of a variable.
  6. Divide and Conquer
      Taking a large problem and dividing it into several little ones, so the task could be more manageable.
  7. else
      The part of an if-statement that is to be executed if the first, or more than one, condition is not met. The code in the else part will be executed then.
  8. Expresssion
      An expression computes a value from a set of operands and operators. The simplest expression is just a literal value. An expression can be made of several expressions too like 'sum(a, b + 2)'.
  9. for-statement
      An iteration method that allows for the management of the control variable to be at the top for easy reading.
  10. range-for-statement
      A for-loop/statement that does the same job as the simplest for-loop (int i; i < size; ++i) but is easier to write. It uses the word 'range' because it is sometimes used as an equivalent to 'sequence of elements'.
  11. Function
      A named sequence of statements.
  12. if-statement
      A conditional statement that executes when its conditional expression is true.
  13. Increment
      Increasing the value of something, like ++i.
  14. Input
      The data taken from the user via a keyboard,mouse,screen, etc or from other programs as well that is used to produced an output(result).
  15. Iteration
      Doing something to a series of elements of a data structure.
  16. Loop
      Going from the start of a sequence of elements to the end.
  17. lvalue
      The 'left value' of an assignment.
  18. Member Function
      A function that is part of an object. Also known as a method in OOP.
  19. Output
      The result of a computation.
  20. push_back()
      The method of vector objects that allows for adding more elements into it at the end of the vector.
  21. Repetition
      Sometimes you need to repeat a certain task, so you need repetition.
  22. rvalue
      The 'right side value' of an assignment.
  23. Selection
      Selecting between different alternatives in programing is done, usually, with if- or switch- statement.
  24. size()
      The method of a vector object that returns its size.
  25. sort()
      A function that sorts a vector, you must pass at least 2 arguments, the starting and ending part.
  26. Statement
      It is the smallest element that expresses some action to be carried out
  27. switch-statements
      A program flow element that allows selection of an outcome based on on a certain input. The input must be of type char, int, or enum but not string. And the values that the input must be compared with must be constants.
  28. Vector
      A sequence of elements that you can access by an index.
  29. while-statements
      A type of iteration that requires a loop(control) variable defined outside the loop.

Review

What is computation?

It is the act of getting an output from an input.

What do we mean by inputs and outputs to a computation? Give examples.

An input could be typing 'yes' into an input field, clicking on a submt button, shinning light into a light receptor, etc,. and the ouput could be a JSON response, sending you to a new website, making an alarm glow to signal the reception of light.

What are the three requirements a programmer should keep in mind when expressing computation?

Correctness, simplicity, and efficiency.

What does an expression do?

It calculates a value.

What is the difference between an expression and a statement, as described in this chapter?

A expression is just a set of operands and operators, or simply a literal, while the statement is an expression terminated by a semicolon.

What is an lvalue? List the operators that require an lvalue. Why do these operators, and not the others, require an lvalue?

The 'left-hand' value of an assignment/compound assignment. These operators are required because they are giving a new value to a named object.

What is a constant expression?

An expression made of constant literals.

What is a literal?

It is a value like '4', 4, "four", true.

What is a symbolic constant and why do we use them?

A constant that is initialized at compilation time. They are useful because they save time when we don't need the program to initialize it (ergo saving running time) and we are sure the value will not be changed at all.

What is a magic constant? Give examples.

A magic constant is a constant(a literal) that just conveys its value rather than what it is. For example if you had a definition for the area of a circle and you used 3.14159 as part of its definition, then the literal 3.14159 is a magic constant. If you were to defined the constant pi before and then use the symbolic pi constant you just defined, then it would be better. This way you know where the literal 3.14149 comes from.

What are some operators that we can use for integers and floating point values?

Addition, substraction, greater than, less than

What are some operators that we can use for integers but not on floating point numbers?

The modulos operator.

What are some operators that can be used for strings?

+=, +, ==

When would a programmer prefer a switch-statement to an if-statement?

When you want optimized code to compare input against a set of unique constants.

What are some common problems with switch-statements?

Forgetting the break at the end of each case, repeating values for the case labels, or using variables instead of constants.

What is the function of each part of the header line in a for-loop, and in what sequence are they executed?

int i=0; The loop/variable declaration and initialization. i < someValue; The conditional statement. i++ The increment operation. The order is, to execute the body body of the for-statement if the loop variable passes the conditional statement starting with the initialized value and then keep increasing the loop variable. Repeat the process until the condition is not passed.

When should the for-loop be used and when should the while-loop be used?

Whenever a loop can be defined with simple loop initializer, condition, and increment operation use for-loops, because it makes code easy to read and understand. Use while-loops when this is not possible.

How do you print the numeric value of a char?

int('a')

Describe what the line char foo(intx) means in a function definition.

It means that a function of type char named foo has a parameter of type int.

When should you define a separate function for a part of a program? List reasons.

When we want abstraction, when we want to reuse that function, and when we want logically separate computation, and ease of testing.

What can you do to an int that you cannot do to a string?

Modulos operations, assignment, etc.

What can you do to a string that you cannot do to an int?

Concatenation

What is the index of the third element of a vector?

2

What does vector<char> alphabet(26) do?

It initializes a vector with 26 char elements with a value of '' because it is a char type.

Describe what push_back() does to a vector?

It adds an element to the end of the vector.

What does vector's member size() do?

It returns the number of elements in the vector.

What makes vectors so popular/useful?

They allow for data to be stored in an ordered mannered.

How do you sort the elements of a vector?

Use the sort() function.
