Terms

  1. Affordability
      Programs need to be cost effective. Requiring a system to use a high end computer might make that product more expensive and the potential customer might not buy it.
  2. Analysis
      We need to know the problem, think about the user's needs and wants. Part of a 4-stage process for developing a program.
  3. Blackboard
      The best design tool. It will allow you to write design ideas, and explain those ideas to your colleagues/friends.
  4. CAD/CAM
      computer-aided design and Computer-aided manufacture.
  5. Communication
      Never try to code alone. Much can be learn by trying to explain your idea to someone else and that person might be able to point out problems/ideas that as a programmer might not think about.
  6. Correctness
      Your programs need to provide correct solutions, otherwise there is no point on using them. It must also be reliable when giving those correct solutions.
  7. Customer
      The users of the program you intend to provide them with.
  8. Design
      Coming up with solutions to a problem, deciding the overall structure of the system and how it communicates with the users. Part of a 4-stage process for developing a program.
  9. Feedback
      We must modify our programs from the experience of the users.
  10. GUI
      Graphical-user interface.
  11. Ideals
      The ideals for programming: correctness, reliability, well-designed, affordable, and maintainable code.
  12. Implementation
      Programming plus testing.
  13. Programmer
      Someone who writes code. Hopefully good code.
  14. Programming
      Expressing solutions to problems in code. It is a tool for expressing solutions so that they can be tested, improved, and used.
  15. Software
      A collection of programs running on some computer.
  16. Stereotype
      The idea that a fat, ugly, loner is the usual programmer.
  17. Testing
      We need to make sure the system works correctly under all circumstances required by systematically trying it out.
  18. User
      The customer using the program.
